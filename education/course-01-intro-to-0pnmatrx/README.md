# Course 01: Introduction to 0pnMatrx

## Overview

This course takes you from zero to a working understanding of 0pnMatrx (OpenMatrix). By the end, you will have built a working plugin, deployed a smart contract, and run your own gateway instance. No blockchain experience required -- just Python and a terminal.

## Prerequisites

- **Python 3.11+** installed and working
- **Basic terminal knowledge** (navigating directories, running commands, reading output)
- A text editor or IDE of your choice
- Git installed for cloning the repository

## What You Will Build

By completing this course, you will have:

1. **A running gateway instance** serving the 0pnMatrx API on port 18790
2. **A working plugin** that adds a custom command to the platform
3. **A deployed smart contract** on Base Sepolia testnet, converted from plain English
4. **SDK integration code** that communicates with the gateway programmatically
