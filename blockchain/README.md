# Blockchain

0pnMatrx is built on blockchain infrastructure. Every capability — smart contracts, identity, payments, governance — is on-chain and verifiable.

## Structure

```
blockchain/
├── contracts/          # Solidity smart contracts
│   ├── Identity.sol
│   ├── Ownership.sol
│   ├── Payment.sol
│   ├── Governance.sol
│   └── Insurance.sol
└── docs/
    ├── components.md   # Blockchain components overview
    └── base-l2.md      # Base L2 integration details
```

## Contracts

All contracts in this directory use `CONFIGURE_BEFORE_DEPLOY` as a placeholder for wallet addresses and configuration values. Replace these with your own values before deploying.
