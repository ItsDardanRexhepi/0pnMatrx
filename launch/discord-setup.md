# 0pnMatrx Discord Server Configuration

Complete Discord server setup guide for the 0pnMatrx platform. All text blocks are ready to copy-paste directly into Discord.

---

## Server Settings

**Server Name:** 0pnMatrx

**Server Icon:** Upload the MTRX logo from the project assets. Use a square version (minimum 512x512px) with the logo centered on a dark background for best visibility on both light and dark Discord themes.

**Server Description (About):**
```
Free, open-source AI agent platform for blockchain. Three AI agents. 30 blockchain services on Base. No token. Free forever.
```

**Community Features:** Enable Community in Server Settings > Enable Community. Set #rules as the Rules channel and #announcements as the Community Updates channel.

## Server Rules

Paste this into your #rules channel or the Rules Screening page under Community settings.

```
0pnMatrx Server Rules

1. Be respectful.
Treat everyone with respect. No harassment, hate speech, personal attacks, or discrimination of any kind. Disagree with ideas, not people.

2. No spam.
Do not flood channels with repeated messages, excessive mentions, or unsolicited DMs to other members. One message is enough.

3. No financial advice.
0pnMatrx is a technology platform. Do not give, solicit, or imply financial advice. Do not make price predictions or investment recommendations. Nothing said here is financial advice.

4. No scams, phishing, or malicious links.
Do not post suspicious links, fake giveaways, airdrop scams, or phishing attempts. Team members will never DM you first asking for funds, seed phrases, or private keys. Report suspicious activity immediately.

5. Keep it constructive.
Criticism is welcome when it is constructive. If something is broken, tell us how. If you have an idea, share it. This is a builder community and we want to make things better together.

6. English is the primary language.
To keep conversations accessible to everyone, please use English in all public channels.

7. No self-promotion outside #showcase.
Share your projects, plugins, and integrations in #showcase. Do not advertise unrelated products, services, or other Discord servers in other channels without team permission.

8. Follow Discord's Terms of Service.
All of Discord's Terms of Service and Community Guidelines apply here. https://discord.com/terms

Breaking these rules will result in a warning, mute, or ban depending on severity. The team reserves the right to take action on behavior not explicitly listed here if it harms the community.
```

## Bot Setup

### MEE6 (Moderation)

**Invite link:** https://mee6.xyz/add

**Setup steps:**
1. Invite MEE6 to the server using the link above.
2. Go to the MEE6 dashboard for your server.
3. Enable the Moderator plugin.
4. Configure auto-moderation:
   - Enable anti-spam: delete messages when a user sends more than 5 messages in 5 seconds. Action: mute for 10 minutes.
   - Enable anti-link: block links in all channels except #showcase, #resources, #general, and #announcements. This prevents phishing in builder channels.
   - Enable banned words: add common scam phrases ("free airdrop", "send ETH to", "validate your wallet", "connect wallet to claim", "DM me for").
   - Enable anti-invite: block Discord server invite links in all channels. Allow only team members to post invites.
5. Configure welcome DM (optional, supplements the system welcome message):
   - Message: "Welcome to 0pnMatrx! Head to #introductions to say hello, and check the pinned message in #general for quick links to get started."
6. Set moderator roles: grant Team role bypass permissions so team members are not affected by auto-moderation.

### Carl-bot (Role Assignment)

**Invite link:** https://carl.gg/invite

**Setup steps:**
1. Invite Carl-bot to the server.
2. Create a #roles channel in the COMMUNITY category (or place the reaction role message in #general).
3. Go to the Carl-bot dashboard > Reaction Roles.
4. Create a new reaction role message with the following text and reactions:

**Reaction role message (paste into the Carl-bot editor):**
```
Pick your role! React below to let us know who you are.

:bust_in_silhouette: Community -- General member, here to learn and participate
:hammer: Builder -- Actively building with 0pnMatrx
:jigsaw: Plugin Developer -- Creating plugins for the platform
```

5. Map the reactions:
   - :bust_in_silhouette: assigns Community role
   - :hammer: assigns Builder role
   - :jigsaw: assigns Plugin Developer role
6. Set the reaction role type to "Multiple" so members can pick more than one role.
7. The Certified Dev and Sponsor roles should be assigned manually by the Team. Do not include them in the reaction role message.
