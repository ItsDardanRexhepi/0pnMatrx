## 2. Ethereum Foundation ESP

### Application URL
https://esp.ethereum.foundation/

### Steps

- [ ] Visit https://esp.ethereum.foundation/
- [ ] Click "Submit an Inquiry" or "Apply"
- [ ] Select category: **Project Grant**
- [ ] Select subcategory: "Community and Education" or "Developer Tooling"
- [ ] Fill in project details using content from `grants/ETHEREUM_FOUNDATION_ESP.md`
- [ ] Emphasize public good nature in every relevant field
- [ ] Provide GitHub URL
- [ ] Enter requested amount: $75,000
- [ ] Describe how the project benefits the Ethereum ecosystem specifically
- [ ] Explain the educational value (reference implementation, example scripts, documentation)
- [ ] Submit inquiry

### Required Information

| Field | Value |
|-------|-------|
| Project Name | 0pnMatrx |
| Requested Amount | $75,000 |
| Category | Project Grant |
| Subcategory | Community and Education / Developer Tooling |
| Public Good | Yes |
| Open Source | Yes (MIT) |
| Token | None |

### Tips

- ESP values projects that benefit the entire Ethereum ecosystem, not just one application. Frame 0pnMatrx as infrastructure that anyone can build on.
- Lead with the public good argument: free, open source, no token, local-first.
- The educational value section is critical. ESP funds projects that teach. The nine example scripts, graceful degradation patterns, and multi-provider architecture are all educational contributions.
- Be specific about what you will deliver with the funds. ESP wants concrete outputs, not vague promises.
- ESP applications often begin with an inquiry that leads to a conversation. The initial submission can be shorter than the full document -- use the document as a reference for follow-up conversations.
- Mention Ollama as the default provider. The fact that users can run the entire platform for free, locally, without an API key, is a strong public good signal.

### Estimated Timeline

- Initial inquiry review: 2-4 weeks
- Clarification questions (common): 1-2 weeks
- Full application review: 4-8 weeks
- Decision: 6-12 weeks from initial inquiry
- Disbursement: milestone-based

## 4. Gitcoin Grants

### Application URL
https://grants.gitcoin.co/ (check for active rounds)

### Steps

- [ ] Visit https://grants.gitcoin.co/
- [ ] Create a Gitcoin Passport (https://passport.gitcoin.co/) if you do not have one
- [ ] Build Passport score by verifying identity stamps (GitHub, Twitter/X, Google, ENS, etc.)
- [ ] Target Passport score of 20+ (required for most rounds)
- [ ] When a relevant round opens (Ethereum Infrastructure, Public Goods, or similar), click "Create a Project"
- [ ] Fill in project details using content from `grants/GITCOIN_GRANTS.md`
- [ ] Upload project logo and banner
- [ ] Provide GitHub URL and NeoSafe address
- [ ] Add project to the relevant active round(s)
- [ ] Share the grant page widely -- Gitcoin uses quadratic funding, so the number of individual donors matters more than donation size
- [ ] Post on social media, developer communities, and relevant forums

### Required Information

| Field | Value |
|-------|-------|
| Project Name | 0pnMatrx |
| Category | Ethereum Infrastructure / Public Goods |
| Website | https://github.com/ItsDardanRexhepi/0pnMatrx |
| GitHub | https://github.com/ItsDardanRexhepi/0pnMatrx |
| Receiving Address | 0x46fF491D7054A6F500026B3E81f358190f8d8Ec5 |
| Gitcoin Passport | Required (score 20+) |

### Tips

- **Quadratic funding means unique donors matter most.** 100 people donating $1 each generates far more matching than 1 person donating $100. Focus outreach on getting many individual supporters.
- Keep the project description accessible and community-facing. Avoid jargon. The Gitcoin audience includes non-technical supporters.
- The quote from Dardan in the application is effective for community connection. Keep it.
- Share the grant page on Twitter/X, Discord communities, Reddit (r/ethereum, r/ethdev), and relevant Telegram groups.
- Update the grant page with progress as the round progresses. Active projects get more attention.
- Consider creating a short (30-60 second) video explaining the project for the grant page.

### Estimated Timeline

- Round duration: typically 2-3 weeks once active
- Matching pool distribution: 2-4 weeks after round ends
- Direct donations: received immediately
- Rounds are periodic -- monitor https://grants.gitcoin.co/ for announcements
